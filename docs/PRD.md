# 교회 설교 음성파일 아카이브 블로그 MVP PRD

## 🎯 핵심 정보

**목적**: Notion을 CMS로 활용하여 교회 설교 음성파일을 아카이빙하고, 성도들이 언제든 설교를 다시 들을 수 있는 블로그 제공
**사용자**: 교회 설교를 다시 듣거나 찾아보고 싶은 성도 및 외부 방문자

---

## 🚶 사용자 여정

```
1. 홈 페이지 (최근 설교 목록)
   ↓ 자동 진입 (사이트 접속 시)

2. 분기 선택
   ├── 특정 글 클릭     → 글 상세 페이지 → 음성 재생 → 완료
   ├── 카테고리 클릭    → 카테고리 페이지 → 글 목록 → 글 상세 페이지
   ├── 달력 메뉴 클릭   → 달력 페이지 → 날짜 클릭 → 글 상세 페이지
   └── 검색창 입력      → 검색 결과 페이지 → 결과 클릭 → 글 상세 페이지

3. 글 상세 페이지
   ↓ 음성 재생 / 본문 열람

4. 완료 → 홈으로 돌아가기 또는 다른 카테고리 탐색
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|---------------|------------|
| **F001** | Notion 글 목록 조회 | Notion API로 발행됨(Status) 상태의 글 목록 가져오기 | 콘텐츠 공급의 핵심 기반 | 홈 페이지, 카테고리 페이지 |
| **F002** | 글 상세 조회 | Notion 페이지 본문 및 첨부 음성파일 조회 | 설교 열람의 핵심 기능 | 글 상세 페이지 |
| **F003** | 음성파일 재생 | 글에 첨부된 음성파일을 브라우저 내 재생 | 서비스의 핵심 가치 | 글 상세 페이지 |
| **F004** | 카테고리(성경 책)별 필터링 | Category 필드 기준으로 글 목록 필터링 | 사용자 탐색 편의성 | 카테고리 페이지 |
| **F005** | 달력 표기 | 발행일(Published) 기준으로 달력에 글 표기 | 날짜별 설교 탐색 지원 | 달력 페이지 |
| **F006** | 제목/본문 검색 | 제목 또는 본문 키워드로 글 검색 | 특정 설교 빠른 접근 | 검색 결과 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |

| **F007** | 반응형 레이아웃 | 모바일/태블릿/데스크톱 대응 | 모바일 사용자 접근성 | 전체 페이지 |

- 태그 기반 필터링 (Tags 필드 활용)

## 📄 페이지별 상세 기능

### 홈 페이지

> **구현 기능:** `F001`, `F007`, `F008` | **메뉴 위치:** 헤더 > 홈

| 항목 | 내용 |
|------|------|
| **역할** | 사이트 진입점 — 최근 발행된 설교 글 목록을 보여주는 랜딩 페이지 |
| **진입 경로** | 사이트 루트 접속 또는 헤더 홈 메뉴 클릭 |
| **사용자 행동** | 최근 설교 목록을 훑어보고 원하는 글 클릭하여 상세 페이지로 이동 |
| **주요 기능** | • Notion API에서 Status=발행됨 글 목록 조회 (ISR 캐싱 적용)<br>• 글 카드 표시: 제목, 발행일, 카테고리(성경 책), 태그<br>• 카테고리 클릭 시 카테고리 페이지로 이동<br>• 검색창 표시 및 검색 입력 시 검색 결과 페이지로 이동<br>• 모바일/데스크톱 반응형 카드 그리드 |
| **다음 이동** | 글 카드 클릭 → 글 상세 페이지 / 카테고리 클릭 → 카테고리 페이지 / 검색 입력 → 검색 결과 페이지 |

---

### 글 상세 페이지

> **구현 기능:** `F002`, `F003`, `F007`, `F008` | **메뉴 위치:** 홈/카테고리/달력/검색 결과에서 글 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 개별 설교 글의 본문 열람 및 음성파일 재생 — 서비스 핵심 페이지 |
| **진입 경로** | 홈 페이지, 카테고리 페이지, 달력 페이지, 검색 결과 페이지에서 글 카드 클릭 |
| **사용자 행동** | 설교 본문을 읽고, 첨부된 음성파일을 재생하여 설교를 청취 |
| **주요 기능** | • Notion 페이지 본문 렌더링 (텍스트, 이미지, 인용문 등 블록 지원)<br>• 음성파일 플레이어 표시: 재생/일시정지, 재생 진행바, 재생 시간 표시<br>• 글 메타 정보 표시: 제목, 발행일, 카테고리, 태그<br>• 뒤로 가기 버튼 (이전 페이지 복귀)<br>• 반응형 레이아웃 |
| **다음 이동** | 뒤로 가기 버튼 → 이전 페이지 / 카테고리 클릭 → 카테고리 페이지 |

---

### 카테고리 페이지

> **구현 기능:** `F001`, `F004`, `F007`, `F008` | **메뉴 위치:** 헤더 > 카테고리

| 항목 | 내용 |
|------|------|
| **역할** | 성경 책(Category)별로 설교 글을 분류하여 탐색하는 필터링 페이지 |
| **진입 경로** | 헤더 카테고리 메뉴 클릭 또는 홈/글 상세 페이지의 카테고리 태그 클릭 |
| **사용자 행동** | 성경 책 목록에서 원하는 카테고리를 선택하고 해당 설교 목록을 탐색 |
| **주요 기능** | • 전체 카테고리(성경 책) 목록 탭/사이드바 표시<br>• 선택한 카테고리의 글 목록 표시 (제목, 발행일, 태그)<br>• 카테고리 전환 시 해당 글 목록으로 즉시 업데이트<br>• 카테고리별 글 수 표시<br>• 반응형 레이아웃 |
| **다음 이동** | 글 카드 클릭 → 글 상세 페이지 |

---

### 달력 페이지

> **구현 기능:** `F001`, `F005`, `F007`, `F008` | **메뉴 위치:** 헤더 > 달력

| 항목 | 내용 |
|------|------|
| **역할** | 월별 달력에 설교 업로드 날짜를 시각적으로 표기하여 날짜별 탐색 지원 |
| **진입 경로** | 헤더 달력 메뉴 클릭 |
| **사용자 행동** | 달력에서 설교가 업로드된 날짜를 확인하고, 날짜 클릭으로 해당 글에 접근 |
| **주요 기능** | • 월별 달력 렌더링<br>• Published 날짜에 글 존재 표시 (점/아이콘 마커)<br>• 글이 있는 날짜 클릭 시 해당 글 목록 팝업 또는 인라인 표시<br>• 이전/다음 월 이동 버튼<br>• 반응형 달력 레이아웃 |
| **다음 이동** | 날짜 클릭 후 글 선택 → 글 상세 페이지 |

---

### 검색 결과 페이지

> **구현 기능:** `F006`, `F007`, `F008` | **메뉴 위치:** 헤더 > 검색

| 항목 | 내용 |
|------|------|
| **역할** | 키워드 기반으로 제목 또는 본문에서 설교 글을 검색하는 전용 페이지 |
| **진입 경로** | 헤더 검색 아이콘 클릭 또는 홈 페이지 검색창에 키워드 입력 후 제출 |
| **사용자 행동** | 검색어를 입력하고, 제목/본문 검색 조건을 선택하여 결과 목록에서 원하는 글 클릭 |
| **주요 기능** | • 검색어 입력창 (자동 포커스)<br>• 검색 범위 선택: 제목 전체 / 본문 전체<br>• Notion API 검색으로 실시간 결과 반환<br>• 검색 결과 목록: 제목, 발행일, 카테고리, 매칭 본문 일부 표시<br>• 결과 없음 상태 표시<br>• 반응형 레이아웃 |
| **다음 이동** | 결과 글 클릭 → 글 상세 페이지 |

## 🗄️ 데이터 모델

### Post (Notion 페이지 매핑)

| 필드 | 설명 | 타입/관계 |
|------|------|----------|
| id | Notion 페이지 ID | string (UUID) |
| title | 설교 제목 | string |
| category | 성경 책 | string (select) |
| tags | 태그 목록 | string[] (multi_select) |
| publishedAt | 발행일 | Date |
| status | 초안 / 발행됨 | string (select) |
| content | 페이지 본문 블록 | NotionBlock[] |

### AudioFile (Notion 첨부파일 매핑)

| 필드 | 설명 | 타입/관계 |
|------|------|----------|
| id | 첨부파일 식별자 | string |
| postId | 연결된 글 ID | → Post.id |
| url | 음성파일 URL | string |
| name | 파일명 | string |
| expiresAt | Notion 임시 URL 만료 시각 | Date |
